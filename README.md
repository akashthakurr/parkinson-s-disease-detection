# 🧠 Parkinson’s Disease Detection using Machine Learning

A Machine Learning project that predicts whether a person has Parkinson’s Disease using biomedical voice measurements and a Support Vector Machine (SVM) model.

---

## 🚀 Overview

This project focuses on early detection of Parkinson’s Disease by analyzing vocal features.  
The model is trained on biomedical voice data and can classify whether a person is healthy or affected.

---

## ⚙️ Tech Stack

- Python 🐍
- NumPy & Pandas
- Scikit-learn
- Support Vector Machine (SVM)
- Jupyter Notebook

---

## 📊 Features

- Data preprocessing and feature selection  
- Feature scaling using StandardScaler  
- SVM model training and evaluation  
- Accurate prediction system  
- Clean and modular ML workflow  

---

## 🧠 Model Details

- Algorithm: **Support Vector Machine (SVM)**
- Kernel: Linear  
- Performance: High accuracy on test dataset  
- Evaluation Metrics: Accuracy Score  

---

## 📁 Dataset

- Biomedical voice measurements dataset  
- Contains multiple acoustic features of voice samples  
- Target variable:
  - `0` → Healthy  
  - `1` → Parkinson’s Disease
